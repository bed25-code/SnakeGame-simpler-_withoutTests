# Snake Game

A simple console Snake game for .NET 9.

## Run

```bash
cd SnakeGame
dotnet run
```

## Controls

Arrow keys to move. You cannot reverse direction into yourself.

## Project structure

- `Program.cs` — console, input, rendering
- `SnakeGameService.cs` — game logic
- `GameState.cs` — state and data
- `Pos.cs` — position
- `Direction.cs` — direction enum
